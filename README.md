# hotel_site
Bu proje, harici kütüphaneler veya karmaşık frameworkler kullanılmadan, sadece saf HTML5 ve CSS3 kullanılarak geliştirilmiş, modern ve duyarlı (responsive) bir otel web sitesi şablonudur.

Projenin çalışan halini aşağıdaki bağlantıdan inceleyebilirsiniz: https://nuray-karabacak.github.io/hotel_site/

Proje Hakkında
Bu web sitesi, bir otelin dijital varlığını oluşturmak için gerekli temel sayfaları içerir. Tasarımda sadelik ve kullanıcı deneyimi ön planda tutulmuştur. JavaScript veya backend bağımlılığı yoktur, bu sayede oldukça hızlı yüklenir ve her türlü sunucuda (veya GitHub Pages'da) sorunsuz çalışır.
